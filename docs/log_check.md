│ 7. Root Cause Analysis: The Accumulation Problem                                                                                                                                                     │
│                                                                                                                                                                                                      │
│ The position went from -0.25 → +16.10 (16.35 net buys, 0 sells) in 61 minutes. Why?                                                                                                                  │
│                                                                                                                                                                                                      │
│ Cause Chain:                                                                                                                                                                                         │
│                                                                                                                                                                                                      │
│ 1. vol_bias collapsed to 0.109 — covariance tracker underestimated realized vol by 9x                                                                                                                │
│ 2. Spreads were too tight — GLFT optimal spread uses σ², so 9x sigma error = ~81x underpriced risk                                                                                                   │
│ 3. Bid ladder levels rested too close to mid — when price dropped, all bid levels swept simultaneously                                                                                               │
│ 4. E[PnL] filter didn't prevent it — the filter evaluates expected PnL per level, not aggregate position risk                                                                                        │
│ 5. Governor addons were position-direction-aware but insufficient — governor_bid was 15 bps at session end, but bids were still placed at deep levels                                                │
│ 6. No sell fills occurred — ask orders were either too far from mid or the market moved only in one direction                                                                                        │
│ 7. Kill-switch only activated at 100% utilization (13:14), 10 minutes after reaching max position at 13:04                                                                                           │
│                                                                                                                                                                                                      │
│ Why the Governor Didn't Save It:                                                                                                                                                                     │
│                                                                                                                                                                                                      │
│ At the end of the session, the governor added 15 bps to bid depth — pushing bids further from mid. But the E[PnL] filter dropped ALL bid levels as negative-EV. So the governor was working, but by  │
│ that point the position was already maxed.                                                                                                                                                           │
│                                                                                                                                                                                                      │
│ The issue is the window between 12:37–12:40: 4.63 contracts bought in 3 minutes with no governor intervention visible. The governor scales additively, but bulk fills from price sweeps arrive       │
│ faster than the quoting cycle can adapt.                                                                                                                                                             │

│ 8. Key Findings & Recommendations                                                                                                                                                                    │

│ CRITICAL (act now)                                                                                                                                                                                   │

│ - vol_bias went 1.00 → 0.12 in one jump, then stayed there for 3 hours                                                                                                                               │

│ C2: Bulk fill vulnerability — ladder swept in single price move                                                                                                                                      │

│ - Current architecture: fill → next quote cycle → spread update. But by then, 5 levels are already filled.                                                                                           │
